- [StepN](https://www.usdtbi.com/gamefi/stepn) - 将 SocialFi 与 GameFi 融合的现象级应用，定义了 Move-to-Earn（跑赚）运动模式。
- [Illuvium](https://www.usdtbi.com/gamefi/illuvium) - 基于 Immutable X 开发的 3A 级开放世界探险与宠物对战角色扮演游戏。
- [Big Time](https://www.usdtbi.com/gamefi/bigtime) - 史诗级多人在线动作 RPG，玩家可以在空间旅行中收集珍稀装备。
- [Pixels](https://www.usdtbi.com/gamefi/pixels) - 像素风农场经营与探索类 Web3 游戏，Ronin 生态近期最火热的交互项目。
- [Ronin Network](https://www.usdtbi.com/gamefi/ronin) - 专门为游戏开发打造的 EVM 兼容区块链，旨在支撑大规模玩家的实时经济活动。
- [Immutable X](https://www.usdtbi.com/gamefi/immutablex) - 游戏领域首选的 Layer 2 扩容方案，提供零 Gas 费的 NFT 铸造与交易体验。
- [Gala Games](https://www.usdtbi.com/gamefi/galagames) - Web3 游戏生态平台，致力于打造真正具备可玩性与经济闭环的区块链游戏。
- [Yield Guild Games (YGG)](https://www.usdtbi.com/gamefi/ygg) - 最大的去中心化链游公会 DAO，为投资者与玩家提供资产投资与收益共享平台。
- [Splinterlands](https://www.usdtbi.com/gamefi/splinterlands) - 基于 Hive 链运行的卡牌策略游戏，拥有庞大且粘性极高的社区玩家群。
- [Alien Worlds](https://www.usdtbi.com/gamefi/alienworlds) - 以太坊、WAX 等多链并发的模拟经济游戏，玩家可通过挖矿获取 Trilium（TLM）资产。
- [Shrapnel](https://www.usdtbi.com/gamefi/shrapnel) - Avalanche 上的 AAA 级模块化射击游戏，提供极高的自定义资产创作空间。

## Cross-chain Bridges (跨链桥与互操作性)

- [Stargate Finance](https://www.usdtbi.com/bridges/stargate) - 基于 LayerZero 构建的全链原生资产流动性传输协议，有效解决跨链滑点与包装代币风险。
- [Across Protocol](https://www.usdtbi.com/bridges/across) - 利用乐观预言机（Optimistic Oracles）机制的高效跨链桥，为以太坊与 L2 提供极速资金桥接。
- [Orbiter Finance](https://www.usdtbi.com/bridges/orbiter) - 专为以太坊 Rollup 网络设计的去中心化跨链桥，支持 zkSync、Arbitrum 等极速低成本互转。
- [Wormhole](https://www.usdtbi.com/bridges/wormhole) - 连接了数十条异构高价值公链（如 Solana 与 Ethereum）的通用消息传递与资产跨链协议。
- [THORChain](https://www.usdtbi.com/bridges/thorchain) - 支持原生资产（如原生 BTC 兑换原生 ETH）的去中心化跨链流动性基础设施。
- [Hop Protocol](https://www.usdtbi.com/bridges/hop) - 高度可扩展的 Rollup 到 Rollup 通用代币跨链桥，流动性提供者的绝佳选择。
- [Synapse](https://www.usdtbi.com/bridges/synapse) - 通用的跨链互操作性网络，支持在各大 EVM 与非 EVM 链之间进行安全的资产传递。
- [Connext](https://www.usdtbi.com/bridges/connext) - 模块化的跨链互操作性协议，可让开发者构建安全的跨链应用（xApps）。
- [Bungee](https://www.usdtbi.com/bridges/bungee) - 最优跨链桥聚合器，智能计算 Gas 费用，确保用户在资金跨链时获取最高效的路由。
- [LayerZero](https://www.usdtbi.com/bridges/layerzero) - 旨在实现超轻量级消息传递的全链（Omnichain）底层互操作性协议。

- [Trality](https://www.usdtbi.com/quant/trality) - 允许使用 Python API 或规则构建器在云端无缝回测并实盘运行算法机器人的平台。
- [OctoBot](https://www.usdtbi.com/quant/octobot) - 模块化架构的开源交易系统，允许投资者灵活装卸各种现成的技术分析策略模块。
